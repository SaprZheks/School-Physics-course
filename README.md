# Введение
Это мой курс по физике. Он идет как часть [сборника моих заметок](https://github.com/SaprZheks/Study-notes), поэтому если вы уже пользуетесь тем репозиторием, курс автоматически встроится в него при запуске ярлыка.
Ниже написаны шаги, которые нужно предпринять, чтобы получить доступ к курсу.

# Инструкция для одиночного использования
1. Выберите место, где локально хотите хранить курс, откройте в этой папке `PowerShell` и выполните

  `git clone https://github.com/SaprZheks/School-Physics-course.git`

<div align="center">
  <img src="readme_guide/1_2.png" alt="1" width="400"/>
</div>

<div align="center">
  <img src="readme_guide/2_2.png" alt="2" width="400"/>
</div>

2. Затем установите программу [Obsidian](https://obsidian.md/download) для просмотра заметок для всех пользователей и нажмите `Запустить` в конце установки (Вернемся к этому окну позже).
3. Переименуйте папку репозитория с `School-Physics-course` на `Курс школьной физики`. Перейдите обратно в репозиторий (папка `Курс школьной физики`).
4. В корне репозитория лежит `Obsidian.bat`. Это файл для запуска Obsidian и автоматического подтягивания всех изменений из этого репозитория. На рабочем столе создайте ярлык, указывающий на `Obsidian.bat`, для красоты можете добавить ему иконку из корня репозитория `Obsidian.ico`.
